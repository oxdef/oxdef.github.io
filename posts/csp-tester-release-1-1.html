CSP Tester 1.1 released

Posted on 09 November 2013 in misc

I am glad to announce the release of CSP Tester 1.1! This extension helps web masters to test web application behaviour with Content Security Policy (CSP) ver. 1.0 implemented.

What's new

  • Directives with wildcard sources are highlighted
  • It is easy to specify source keywords via special checkboxes
  • Small user interface improvements
