CSP Tester 2.0 released

Posted on 09 May 2016 in misc

Today I'm announcing the release of CSP Tester 2.0. This brings with it a few new features as well as bug fixes. The main focus of this release was Content Security Policy Level 2 support.

In addition to CSP2 support, the following changes have been made:

  • Simple mode form is dynamic now. Base form contains only a small number of directives.
  • Added a checkbox for none source
  • Replaced text labels with icons. It made possible to increase length of directive sources input fields
  • Removed legacy code from background script

New version is already in Chrome Web Store.
